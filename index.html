SOAL A: Mengapa menggunakan CSS dan JavaScript?

CSS digunakan untuk mengatur presentasi (tampilan dan gaya visual), memisahkan gaya dari struktur HTML untuk keterawatan dan konsistensi yang lebih baik. JavaScript digunakan untuk menambahkan interaktivitas, perilaku dinamis, dan memanipulasi elemen di halaman (DOM).

SOAL B: Konsep dan Tujuan Responsive Web Design (RWD)

Konsep RWD adalah membuat situs web beradaptasi secara mulus dengan berbagai ukuran layar (desktop, tablet, smartphone) menggunakan teknik seperti *media queries*, *flexible grids*, dan *flexible images*. Tujuan utamanya adalah memberikan pengalaman pengguna yang optimal dan konsisten di semua perangkat.

SOAL C: Kombinasi CSS, JavaScript, dan RWD

Ketiganya saling melengkapi: HTML menyediakan struktur; CSS + RWD memastikan struktur tersebut memiliki tampilan yang adaptif dan rapi di berbagai ukuran layar melalui *media queries* dan tata letak fleksibel (Flexbox/Grid); JavaScript menambahkan interaksi dinamis (seperti fitur sembunyikan/tampilkan konten ini dan pengubahan mode) yang tetap berfungsi di lingkungan responsif yang diciptakan oleh RWD.
